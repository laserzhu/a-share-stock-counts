# A股涨跌家数抓取工具

这是一个简单的 Python 脚本，用于抓取 A 股当前的上涨家数和下跌家数。

## 运行规则
1. **实时抓取**：抓取 A 股当前的上涨家数和下跌家数。
2. **非交易日处理**：如果运行当天是非交易日（周六、周日或法定节假日），脚本将自动显示上一个交易日的统计数据。
3. **手动运行提示**：手动运行脚本时，会首先输出上述运行规则。

## 如何运行
1. 确保已安装 Python 3。
2. 安装依赖库：
   ```bash
   pip install requests
   ```
3. 运行脚本：
   ```bash
   python main.py
   ```

## 数据来源
数据来源于东方财富网公开 API。
